= NASA Mars Rover Photos API PHP Wrapper

== Usage
.Sample usage
[source, php]
----
require_once('path/to/src/include.php');

$mars = new MarsRoverPhoto(
  apiKey: 'DEMO_KEY',
  rover: 'curiosity',
);

$photos = $mars->sol(100)->camera('NAVCAM')->get();
----
